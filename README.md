 # Banking Risk Analytics

  End-to-end risk analytics projects built with Python, SQL, and GCP.

  ## Projects

  ### 1. Customer Risk Scoring
  SQL Server-based customer risk scoring pipeline using transaction, loan, and sanctions data.

  ### 2. Fraud Detection on GCP
  Real-time fraud detection pipeline on Google Cloud Platform.
  - **Cloud Storage** - Raw data ingestion
  - **Dataflow** - Apache Beam ETL pipeline
  - **BigQuery** - Analytics + ML-based fraud scoring (99.6% AUC)
  - **Looker Studio** - Monitoring dashboards

 ### 3. Strategic Recon
  Automated reconciliation and sanctions screening pipeline for banking data.
